Fantasy Football Live Draft Helper

JavaFX, Personal Program

Java/JavaFX, Senior Capstone Group Project


Own project to streamline live fantasy football draft. Program that parses a text file of the top 300 Fantasy Football projected players, and populates a table by the correct draft rank. Full Stack development. When you begin the draft the contestants playing will be displayed with current pick and next pick going in a snake order. A timer begins counting down from 60 seconds, the user selects a player and selects draft, removing the player from the available players pool and updating the current and next pick while restarting the timer. This is a personal program but can easily be converted to a commercial product where a user enters the different contestants playing
